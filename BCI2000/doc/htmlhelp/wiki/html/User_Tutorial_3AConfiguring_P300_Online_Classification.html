Configuring P300 Online Classification

User Tutorial

This tutorial step assumes that you have identified P300 parameters in an initial session. Now you are going to create a subject-specific parameter configuration for on-line spelling.

Starting up BCI2000

Start BCI2000 using the appropriate batch file at batch/P3Speller_<YourAmplifier>.bat. In you have not done so yet, you might consider creating a link to this file on the desktop.

Subject-Specific Parameters

In this step, we start with a generic parameter file, and add subject-specific information to construct a full parameter file for the subject.

  • In the configuration window, click "Load Parameters", and load the system-specific parameter file saved previously at parms/fragments/<YourSystem>.prm.
  • Again, click "Load Parameters" to load parms/p3_tutorial/P3Speller.prm.
  • When you have a separate monitor for experimenter and subject, load the parameter file at parms/fragments/stdlib/DualMonitor.prm.
    • In your system's display properties configuration, make sure that the subject's monitor is configured to be located to the right of the main monitor.
    • Make sure the WindowLeft parameter matches the main monitor's actual pixel width.
    • You may need to adapt WindowTop, WindowWidth, and WindowHeight parameters as well; click "Set Config" to try the effect of your changes.
  • Go to the Storage tab, and enter the subject's name or ID into the SubjectName parameter.
  • Into the SubjectSession parameter, enter 002.

Configuring the Classifier

Subject-specific electrode location and latency are part of the classifier's configuration. Here, we assume that your list of candidate features from the previous step has 3 entries, corresponding to the number of rows in the Classifier parameter.

  • Go to the Classifier parameter on the Filtering tab; there, click the "Edit Matrix" button associated with the Classifier parameter.
  • Set Number of columns to 4, and Number of rows to 3 (or the number of entries in your feature list); then, click Set new matrix size to apply your changes.
  • In the first column, labeled input channel, enter the location of the classification electrode.
  • In the second column, labeled input element (bin), enter the temporal offset of the best classification time point, immediately followed with ms, as in 300ms.
  • In the third column, enter 1 as output channel.
  • In the fourth column, enter the number you noted in the previous step: +1 for a positive deflection, -1 for a negative one.
  • Repeat the last steps for the remaining rows; in the second column, enter the temporal offset of the second and third best classification time point.
  • Finally, save your configuration to parms/subjects/p3_speller/<Subject>002.prm, or wherever you find appropriate.

Next Step

In the next step, you will learn how to perform a P300 spelling session using the configuration created in the present step.
